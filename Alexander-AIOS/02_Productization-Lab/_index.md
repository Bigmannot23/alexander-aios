# Productization Lab Index

## What belongs here

Candidate product ideas *extracted from proven internal workflows* — things
that have already worked repeatedly inside this repo (a skill, a script, a
template, a pattern in `01_ClaudeOps/`) and might be worth packaging for
others. Weekly memo cadence, never a daily priority.

Anything not yet proven internally. This lab never designs abstract-first —
see `Master-Blueprint-V1.md` §15 ("ClaudeOps template repo — extract after
`01_ClaudeOps/` is proven, never abstract-first"). No runtime/shippable
code lives here either; a productized idea graduates to its own repo
later, the same way `entrylens/` and `claudeops-template/` do.
