# Claude Surface Routing

- Claude Project Chat: strategy, synthesis, review, one-off reasoning. Not
  durable — copy anything worth keeping into the repo.
- Claude Code: repo edits, skills, scripts, tests, commits, Git.
- Plan Mode stays ON for any multi-file edit or any change that produces a
  durable artifact (doctrine, rules, decision logs, skills).
- Not in use yet, by design: Routines, Dispatch, MCP/connectors, subagent
  swarms beyond what's already approved, hooks. Don't reach for any of
  these before their gate is met — see `01_ClaudeOps/Hooks/README.md`,
  `01_ClaudeOps/Routines/README.md`.
- No trading, account, or payment task runs on any surface — automated or
  not — beyond what `entrylens-trading-safety.md` already forbids.
