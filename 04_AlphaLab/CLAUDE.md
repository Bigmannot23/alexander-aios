# CLAUDE.md — AlphaLab router

Scoped router for `04_AlphaLab/` only. The root [`/CLAUDE.md`](../CLAUDE.md)
and [`/00_System/Master-Blueprint-V1.md`](../00_System/Master-Blueprint-V1.md)
still govern everything below — this file adds AlphaLab-specific emphasis,
it does not replace them.

## Mission

Bounded trading research, review, and mistake-mining. Creative but
non-executing. Markdown knowledge work only — no executing code in this
folder.

## Non-negotiable safety rules (AlphaLab-specific emphasis)

- Every output here is **post-session only** or **research only** (see
  `00_System/Safety-Policy.md` safety classifications) — never live trade
  commentary.
- Never recommend a trade, symbol, contract, size, stop, or target, even
  hypothetically or as an "example."
- Never claim edge, probability, expected return, or win rate for any
  setup concept.
- Setup concepts, no-trade filters, and mistake-taxonomy entries must be
  written as *observable, falsifiable conditions* — not advice, not
  opinions about quality.
- Anything proposed for EntryLens is a *candidate* only
  (`Predicate-Candidates/_index.md`) and crosses over only through the
  promotion lock in `00_System/Promotion-Rules.md` — never a direct edit
  to EntryLens.

## Allowed outputs

Non-advisory session reviews, mistake-taxonomy entries, setup *concepts*,
no-trade *filter* candidates, replay drills, review rubrics, deterministic
predicate candidates (staged only), education notes.

## Forbidden outputs

Trade calls, symbol/contract picks, sizing or stop/target guidance, edge
or probability claims, anything that reads as "I'd take this," broker/
account/live-data connections.

## Automation ceiling

Manual review only. No live commentary system exists. No hooks, routines,
or scheduled trading-adjacent task will ever be created here — see
`Master-Blueprint-V1.md` §15.
